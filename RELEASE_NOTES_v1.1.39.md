# Recipe Helper v1.1.39

## Plan updates

- Gatherable and Collectable plans now show both Save new plan and Update plan controls, matching every other plan section.
- Update plan remains visible when no plan is loaded and explains how to enable it.
- Updating a plan section now changes only that section, preserving its recipes, gatherables, collectables, and independent ingredient snapshots.

## Interface polish

- Rounded action buttons, overlay section headers, and table headers now match the main folder style.
- Tooltip backgrounds follow their rounded accent outline without square corners showing through.
- Plan labels and saved-folder hover headings use the brighter interface accent for better readability.
