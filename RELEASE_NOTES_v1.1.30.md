# Recipe Helper v1.1.30

## What's New

- Added a `Stop After Craft` button to the Artisan progress overlay so queued crafting can end cleanly after the current craft finishes.
- Improved the overlay status text so pre-craft warnings no longer conflict with an actively crafting final recipe.
- Preserved the original crafting order in the progress list with numbered steps, making repeated pre-crafts much easier to follow.
- Closing the Recipe Helper queue now closes the in-game crafting windows after Artisan stops or the queue completes.
