# Recipe Helper v1.1.35

## What's New

- Added inline recipe and gatherable level labels so search results show the required crafting or gathering level next to the job abbreviation.
- Added richer unlock hover details for gatherables and collectables, including folklore books, required tools or items when game data exposes them, and master recipe books for crafted recipes.
- Extended fishing support so FSH items can show level and folklore details where the fishing sheets provide them.
- Replaced marketboard hovers for recognized Ishgard Restoration items with Restoration-specific tooltip content, including phase and Skybuilders' Scrip rewards.
- Removed deprecated Skybuilders items from search when they no longer map to live Ishgard Restoration data.
