# Recipe Helper v1.1.18

## What's New

- Added the new Artisan progress popup with current, queued, and completed recipe tracking plus a live session timer.
- Improved `Gwen's Dream` so it can start crafting even when materials are already in inventory, and tightened AutoRetainer pause and resume handling.
- Expanded theme customisation with hex colour entry, separate accent and section text colours, saved-plan text colours, and built-in blue, pink, purple, green, and orange presets.
- Unified button styling across the plugin so craft actions, settings, debug tools, and overlays all follow the same theme rules.
- Improved scaling and layout behaviour, including better support for smaller interface sizes and placing the `Selected Recipes` action buttons inside the section.
- Added stronger saved-plan folder handling with parent and subfolder organisation plus clearer folder colouring options.
